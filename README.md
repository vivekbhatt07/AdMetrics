# ADMETRICS

## Live Link - https://admetrics.netlify.app/

A social media website where you can connect with your friends!

---

## INSTALLATION

```
1. git clone https://github.com/vivekbhatt07/AdMetrics
2. cd AdMetrics
3. npm install
4. npm run dev
```

## Built with -

- React JS
- React Router v6
- Tailwind CSS
- Material UI
- React Responsive
- Material UI icons
- Lottie React
